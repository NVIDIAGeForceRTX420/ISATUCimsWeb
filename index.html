ðŸ“Œ SQL Queries

Below are some key SQL queries used in the system:


-- Show all documents with authors
SELECT d.title, r.fname, r.lname
FROM document d
JOIN researcher r ON d.researcher_id = r.researcher_id;

-- List duplicate documents
SELECT d.title, a.similarity_score
FROM analysis a
JOIN document d ON a.document_id = d.document_id
WHERE a.is_duplicate = TRUE;

-- Count of documents by academic level
SELECT academic_level, COUNT(*) FROM document GROUP BY academic_level;

-- Transaction logs
SELECT t.log_id, d.title, t.status, t.timestamp
FROM transaction_log t
JOIN analysis a ON t.analysis_id = a.analysis_id
JOIN document d ON a.document_id = d.document_id;
